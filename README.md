## 🚀 프로젝트 목록

### 1. Smart Habit Tracker
**기술 스택**: React · Python Flask · MySQL · Chart.js

개인의 하루 목표 달성률을 분석하여 시각화해주는 습관 관리 웹 서비스입니다.

**주요 기능**:
- 습관 추가/삭제
- 일일 완료 체크
- 연속 일수 추적
- Chart.js를 활용한 달성률 시각화

**데모**: [habit-tracker.html](./habit-tracker.html)

---

### 2. Recipe Explorer
**기술 스택**: React · Python · REST API

냉장고 속 재료를 기반으로 요리 레시피를 추천해주는 웹 서비스입니다.

**주요 기능**:
- 보유 재료 입력
- 재료 기반 레시피 검색
- 매칭 레시피 표시
- 반응형 카드 레이아웃

**데모**: [recipe-explorer.html](./recipe-explorer.html)

---

### 3. Study Time Analyzer
**기술 스택**: Python · MySQL · Chart.js

공부 시간을 자동 분석해 대시보드로 시각화하는 학습 분석 서비스입니다.

**주요 기능**:
- 공부 시간 기록
- 통계 카드 (총 시간, 평균, 연속 일수)
- 일별 공부 시간 라인 차트
- 과목별 공부 시간 도넛 차트

**데모**: [study-analyzer.html](./study-analyzer.html)
